Welcome to the xVector MMORPG Engine documentation!
===================================================

The xVector MMORPG Engine is an open source toolkit for creating 2D massively
multiplayer online role playing games. xVector gives you everything you need
to quickly and easily create your own virtual world.  No knowledge of
programming is necessary; all the tools you need are provided out of the box.
If you do have programming experience, xVector's Python scripting API allows
you to easily extend the game engine to your heart's content.
